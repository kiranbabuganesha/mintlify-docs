# Contacts Level Attributes
| Attribute                   | Type    | Format | Example              | Description                                       |
|-----------------------------|---------|--------|----------------------|---------------------------------------------------|
| `id`                        | string  |        | `12345`              | Unique identifier associated with a record.        |
| `firstName`                 | string  |        | `John`               | First name of an individual.                       |
| `lastName`                  | string  |        | `Doe`                | Last name or surname of an individual.             |
| `jobTitle`                  | string  |        | `Software Engineer`  | Job or professional title/position.                |
| `seniority`                 | string  |        | `Senior`             | Level of seniority or rank within an organization. |
| `linkedInUrl`               | string  |        | `https://www.linkedin.com/in/johndoe` | URL pointing to an individual's LinkedIn profile.  |
| `department`                | string  |        | `Marketing`          | Department or division within an organization.     |
| `phone`                     | string  |        | `+1234567890`        | Phone number associated with an individual/contact.|
| `country`                   | string  |        | `United States`      | Country of residence or origin.                    |
| `state`                     | string  |        | `California`         | State or region within a country.                  |
| `city`                      | string  |        | `San Francisco`      | City where an individual resides.                  |
| `account.id`                | string  |        | `987654321`          | Unique identifier associated with the account.     |
| `account.name`              | string  |        | `ABC Corporation`    | Name of the account.                               |
| `account.domain`            | string  |        | `example.com`        | Domain associated with the account.                |
| `account.websiteUrl`        | string  |        | `https://www.example.com` | URL of the account's website.              |
| `account.linkedinUrl`       | string  |        | `https://www.linkedin.com/company/example` | URL pointing to the account's LinkedIn page.       |
| `account.twitterUrl`        | string  |        | `https://twitter.com/example` | URL pointing to the account's Twitter profile.     |
| `account.facebookUrl`       | string  |        | `https://www.facebook.com/example` | URL pointing to the account's Facebook page.  |
| `account.industry`          | string  |        | `Technology`         | Industry or sector associated with the account.    |
| `account.employeeCount`     | integer | int32  | `100`                | Number of employees working for the account.       |
| `account.annualRevenue`     | string  |        | `5 Mil`              | Annual revenue associated with the account.        |
| `account.latestFunding`     | string  |        | `2023-01-15`         | Information about the latest funding of the account|
| `account.totalFunding`      | integer | int64  | `10 Mil`             | Total funding received by the account.             |
| `account.state`             | string  |        | `California`         | State or region associated with the account.       |
| `account.city`              | string  |        | `San Francisco`      | City associated with the account.                  |
| `account.technologies`      | array   |        | `Chat GPT, Sprouts`  | Array of technologies associated with the account. |
