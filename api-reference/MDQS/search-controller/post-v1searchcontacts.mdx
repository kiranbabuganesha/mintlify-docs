## Contact Level Attributes
| Attribute        | Type       | Format           | Example                             | Description                                  |
|------------------|------------|------------------|-------------------------------------|----------------------------------------------|
| id               | String     | Text             | `IM001`                              | Unique identifier for Iron Man                |
| database         | String     | Text             | `AvengersDB`                        | Database storing Iron Man`s information      |
| name             | String     | Text             | `Tony Stark (Iron Man)`              | Full name and superhero alias                |
| email            | String     | Email format     | `tony.stark@starkindustries.com`   | Email address of Tony Stark                  |
| linkedinUrl      | String     | URL              | `https://www.linkedin.com/in/tony-stark-avengers`       | LinkedIn profile URL of Tony Stark  |
| twitterUrl       | String     | URL              | `https://twitter.com/iron_man`      | Twitter profile URL of Iron Man              |
| phone            | String     | Phone format     | `+1234567890`                       | Phone number of Tony Stark                   |
| account.domain   | String     | Text             | `starkindustries.com`               | Domain associated with Stark Industries       |
| departments      | Array      | Text Array       | [`Avengers`, `Stark Industries`]    | Organizations Tony Stark is associated with  |
| jobTitle         | String     | Text             | `CEO, Engineer, Superhero`           | Job roles and superhero title                |
| designation      | String     | Text             | `Iron Man`                          | Specific designation as Iron Man             |
| city             | String     | Text             | `New York City`                     | City of residence                            |
| state            | String     | Text             | `NY`                                | State or region                              |
| country          | String     | Text             | `USA`                               | Country of residence                         |
| emailStatus      | String     | Text             | `verified`                            | Status of Tony Stark`s email account         |
| phoneStatus      | String     | Text             | `verified`                            | Status of Tony Stark`s phone                 |
| targetProfile    | Boolean    | True/False       | `true`                                | Indicates if Tony Stark`s profile is targeted|
| page             | Integer    | Number           | `1`                                   | Current page number in database              |
| size             | Integer    | Number           | `20`                                  | Number of records displayed per page         |
| totalElements    | Integer    | Number           | `500`                                 | Total number of records for Iron Man         |
