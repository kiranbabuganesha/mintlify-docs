## Account Level Attributes

| Attribute        | Type       | Format         | Example                         | Description                              |
|------------------|------------|----------------|---------------------------------|------------------------------------------|
| id               | String     | Text           | `ABC123`                        | Unique identifier for the record         |
| database         | String     | Text           | `CompaniesDB`                   | Database storing company information     |
| name             | String     | Text           | `Stark Industries`               | Name of the company                      |
| logoUrl          | String     | URL            | `https://www.example.com/logo`  | URL of the company's logo                |
| websiteUrl       | String     | URL            | `https://www.starkindustries.com`| Website URL of the company               |
| linkedinUrl      | String     | URL            | `https://www.linkedin.com/company/stark-industries`| LinkedIn profile URL       |
| twitterUrl       | String     | URL            | `https://twitter.com/stark_industries`| Twitter profile URL              |
| facebookUrl      | String     | URL            | `https://facebook.com/starkindustries`| Facebook profile URL          |
| industry         | String     | Text           | `Technology`                    | Industry or sector of the company       |
| rawAddress       | String     | Text           | `123 Main Street, New York, USA` | Full address of the company             |
| city             | String     | Text           | `New York`                      | City where the company is located       |
| streetAddress    | String     | Text           | `123 Main Street`               | Street address of the company           |
| state            | String     | Text           | `NY`                            | State or region                          |
| country          | String     | Text           | `USA`                           | Country where the company is situated   |
| postalCode       | String     | Text           | `10001`                         | Postal code or ZIP code                  |
| foundedYear      | Integer    | Number         | 1939                            | Year when the company was founded       |
| employeeCount    | String     | Text           | `5000+`                         | Approximate count of employees           |
| retailLocation   | String     | Text           | `Yes`                           | Indicates if the company has retail locations|
| domain           | String     | Text           | `starkindustries.com`           | Company's domain name                    |
| targetProfile    | Boolean    | True/False     | `true`                            | Indicates if this profile is a target    |
| page             | Integer    | Number         | `1`                               | Current page number in pagination        |
| size             | Integer    | Number         | `20`                              | Number of records displayed per page     |
| totalElements    | Integer    | Number         | `500`                             | Total number of company records          |
